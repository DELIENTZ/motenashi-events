# 読者投稿フォームの作り方（Googleフォーム）

読者や主催者から「このイベント載せて」を受け取るフォームを、無料のGoogleフォームで作ります。
所要：10分くらい。

---

## STEP 1：フォームを新規作成

1. https://forms.google.com/ を開く（会社のGoogleアカウントでOK）
2. 「空白のフォーム」を作成
3. タイトル：`もてなし広場 イベント情報の提供`
4. 説明欄（任意）：
   > もてなし広場のイベント情報をお寄せください。掲載は無料です。確認のうえ掲載します。

---

## STEP 2：質問を追加（下の項目をそのまま）

| # | 質問 | 形式 | 必須 |
|---|------|------|------|
| 1 | イベント名 | 記述式（短文） | ◯ |
| 2 | 開催日（初日） | 日付 | ◯ |
| 3 | 終了日（複数日のとき） | 日付 | － |
| 4 | 開催時間（例：10:00〜15:00） | 記述式（短文） | － |
| 5 | 会場 | ラジオボタン：`もてなし広場` / `庁舎前広場` / `その他` | ◯ |
| 6 | カテゴリ | ラジオボタン：`マルシェ・市` / `グルメ` / `祭り` / `音楽` / `その他` | － |
| 7 | イベント内容・紹介 | 段落 | － |
| 8 | 入場料（例：入場無料） | 記述式（短文） | － |
| 9 | 公式サイト・SNSのURL | 記述式（短文） | － |
| 10 | 主催者・お問い合わせ先 | 記述式（短文） | － |
| 11 | お名前（提供者・任意） | 記述式（短文） | － |

> 画像（チラシ）添付を付けたい場合は「ファイルのアップロード」を足せますが、投稿者にGoogleログインを求めるため、まずは無し（URLで代用）がおすすめです。

---

## STEP 3：回答をスプレッドシートに集める

1. フォーム編集画面の上部「回答」タブ →「スプレッドシートにリンク」
2. 新しいスプレッドシートを作成 → 投稿がここに自動で溜まります

---

## STEP 4：サイトのボタンに接続

1. フォーム右上の「送信」→ リンク（🔗）アイコン → 「URLを短縮」にチェックして **URLをコピー**
2. `motenashi-events/assets/app.js` の先頭付近：
   ```js
   const SUBMIT_FORM_URL = ""; // ← この "" の中にコピーしたURLを貼る
   ```
   例：
   ```js
   const SUBMIT_FORM_URL = "https://forms.gle/xxxxxxxx";
   ```
3. 保存 → GitHub Desktopで Commit → Push
4. 反映後、サイトの「投稿フォーム準備中」ボタンが **「情報を提供する（無料）」** に変わり、押すとフォームが開きます

> URLが空のうちは、ボタンは押せない「準備中」表示になります（死リンクにならない設計）。

---

## STEP 5：届いた投稿をサイトに載せる

1. スプレッドシートで内容を確認
2. 問題なければ `data/events.json` に1件追加（README STEP5のテンプレート参照）
   - ※ 投稿文そのままでなく、事実を確認して自分の言葉でまとめると安心・SEO的にも有利
3. Commit → Push で本番反映

---

## 将来の自動化（任意）
スプレッドシートの内容を自動で `events.json` に取り込む仕組みも後から追加できます（GitHub Actions＋シート公開API）。まずは手動運用で十分です。
